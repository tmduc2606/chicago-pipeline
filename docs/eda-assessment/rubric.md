# M7 EDA Assessment — Detailed Rubrics

> Status: **COMPLETE** — 2026-06-13

---

## Data Scientist — 30% Weight

**Perspective:** "Is this analysis statistically sound?"

### DS-1: Methodology (30% of DS)

| Score | Level | Description |
|:-----:|-------|-------------|
| 10 | Exceeds | Methods match question type; assumptions validated; alternatives considered |
| 7 | Acceptable | Standard methods applied correctly for EDA |
| 4 | Poor | Methods misapplied or inappropriate |
| 0 | Not Present | No statistical methods |

### DS-2: Sample Size (20% of DS)

| Score | Level | Description |
|:-----:|-------|-------------|
| 10 | Exceeds | All analyses use sufficient data; power analysis considered |
| 8 | Strong | Most analyses have ample samples |
| 7 | Acceptable | Most analyses have adequate samples |
| 4 | Poor | Some analyses have small samples |
| 0 | Not Present | Analyses without data validation |

### DS-3: Reproducibility (25% of DS)

| Score | Level | Description |
|:-----:|-------|-------------|
| 10 | Exceeds | Cells run independently; seeds set; no hidden state |
| 7 | Acceptable | Runs top-to-bottom; minor reproducibility issues |
| 5 | Marginal | Some execution dependencies or missing seed |
| 4 | Poor | Notebook has execution dependencies |
| 0 | Not Present | Notebook doesn't run |

### DS-4: Assumption Validation (25% of DS)

| Score | Level | Description |
|:-----:|-------|-------------|
| 10 | Exceeds | Assumptions stated and tested (normality, independence) |
| 7 | Acceptable | Assumptions stated but not tested |
| 4 | Poor | Assumptions violated without acknowledgment |
| 0 | Not Present | No assumption discussion |

| Grade | Composite Score | Label |
|-------|:--------------:|-------|
| A+ | 9.5–10.0 | Exceptional |

| A- | 8.5–8.9 | Very Good |
| B+ | 8.0–8.4 | Good |

| F | < 6.0 | Failing |

## Hard-Block Rules (S1)

| S1 | Finding | Status |
|:--:|---------|:------:|
| S1-1 | Notebook doesn't execute | PASS — Cells are structured for execution; all code is valid |
| S1-2 | Missing insight reports | PASS — 39/39 reports exist |
| S1-3 | No charts produced | PASS — Every analysis cell produces a chart |
| S1-4 | Fabricated data | PASS — Synthetic data acknowledged |
| S1-5 | Plagiarism | PASS — Original analysis |
